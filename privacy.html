Rentex SL+ Assistant

Privacy Policy

Last updated: 2026-07-13

Rentex SL+ Assistant is a browser extension built for internal use with RTPro (the rental & inventory management system hosted at *.rtprosl.com). It adds a side panel that shows the accessory checklist and inspection steps for the item currently under QC, and lets techs attach, detach, and pass inspection steps directly from that panel.

Short version

This extension does not collect, store, or transmit any data to its developer or to any third party. Everything it does happens inside your own already-authenticated RTPro session, talking only to *.rtprosl.com.

What it accesses, and why

Session / authentication data

The extension reads the bearer token and session context headers (location, workstation, hub, etc.) that RTPro's own web app already attaches to its own requests. This is used only to make additional same-origin API calls to *.rtprosl.com as the currently logged-in user — for example, fetching an accessory list or attaching a scanned barcode. This data is kept in memory for the life of the browser tab/panel and is never written to disk, never logged, and never sent anywhere other than *.rtprosl.com.

RTPro application data

To build the checklist, the extension reads RTPro's own API responses — equipment, accessory, inspection, and order data — directly from *.rtprosl.com. This data is same-origin, belongs to the same authenticated session it was read from, and is displayed only to the user that session belongs to.

Local preferences

Two small UI preferences — whether completed checklist items are hidden, and whether audio scan-feedback tones are muted — are saved locally in your browser via the standard chrome.storage.local API, so they persist between sessions. This is not personal or business data, and it never leaves your browser.

What this extension does not do

  • It does not send data to the developer, an analytics service, or any server other than *.rtprosl.com.
  • It does not use cookies, trackers, or fingerprinting.
  • It does not sell or share data with third parties.
  • It does not run a background service that phones home — there is no external backend.
  • It does not operate on any site other than *.rtprosl.com.

Permissions

  • Host access to *.rtprosl.com — required to read RTPro's own session/network traffic and make same-origin API calls on the user's behalf.
  • Side panel — used to render the extension's UI docked beside the RTPro tab.
  • Storage — used only for the two local preferences described above.
